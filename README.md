# Movie Website

A movie website that showcases movies and tv series.

## Run Locally

Clone the project

```bash
  git clone https://github.com/Singatha/movie-website.git
```

Go to the project directory

```bash
  cd movie-website
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
